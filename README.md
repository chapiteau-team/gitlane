# Gitlane

Gitlane is a Git-native task tracker.
All tracker data is stored as Markdown files in the repository.

The design goals are:

- Human-readable data
- AI-friendly structure
- Deterministic behavior
- Full version history through Git
- Validation-first workflows
- Configurable workflow states and transitions
